OpenCleaner
First launch

Full Disk Access required

OpenCleaner scans developer caches and build artifacts that live in protected locations (e.g. ~/Library). Without Full Disk Access, results will be incomplete.

Status: Not granted

We’ll automatically detect when permission is granted. You can keep this window open.

Annotation: Primary button deep-links to System Settings → Privacy & Security → Full Disk Access. Continue is disabled until permission is detected.
Steps
  1. In System Settings, enable Full Disk Access for OpenCleaner.
  2. Return here. Status will update automatically.
  3. OpenCleaner starts a first scan after access is granted.
Keyboard: ⌘ , Settings (disabled on first run), ⌘ . Cancel.
